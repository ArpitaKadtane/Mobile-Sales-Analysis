# Mobile Sales Analysis Dashboard

## Overview
This project is an interactive Power BI dashboard developed to analyze mobile sales performance, customer behavior, and product trends. The dashboard provides meaningful business insights through dynamic visualizations and KPI tracking.

## Features
- Sales performance analysis by brand and region
- Monthly and yearly sales trend visualization
- Customer rating and transaction analysis
- Quantity sold and revenue tracking
- Interactive filters and slicers for better insights
- KPI cards for quick business overview

## Tools & Technologies
- Power BI
- DAX
- Data Cleaning & Transformation
- Data Visualization

## Insights Generated
- Identified top-performing mobile brands
- Analyzed customer purchasing trends
- Compared sales across different cities/regions
- Tracked revenue and quantity sold over time

## Project Files
- `Mobile_Sales_Dashboard.pbix` → Power BI dashboard file
- `dashboard_screenshot.png` → Dashboard preview image

## How to Use
1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Explore the dashboard using filters and slicers
